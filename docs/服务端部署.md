# 服务端部署（阿里云）

## 部署步骤

```bash
# 1. 进入服务端目录
cd /Users/jjwang/Downloads/wangboworkspace/print-miniprogram-admin

# 2. 本地备份
ssh root@39.104.59.201 "cp /opt/print-miniprogram-admin/server.js /opt/print-miniprogram-admin/server.js.backup.$(date +%Y%m%d-%H%M%S)"

# 3. 上传 server.js（重构后只需上传入口文件）
scp server.js root@39.104.59.201:/opt/print-miniprogram-admin/

# 4. 如果改了路由/服务，还要上传 src/ 目录
scp -r src/ root@39.104.59.201:/opt/print-miniprogram-admin/

# 5. 重启 PM2
ssh root@39.104.59.201 "pm2 restart print-admin"

# 6. 检查状态和日志
ssh root@39.104.59.201 "pm2 status print-admin"
ssh root@39.104.59.201 "pm2 logs print-admin --lines 30"
```
