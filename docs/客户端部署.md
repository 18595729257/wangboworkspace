# 客户端部署

## 推荐方式：PM2 管理（替代 nohup）

```bash
# 1. 安装 PM2
npm install -g pm2

# 2. 进入客户端目录
cd /Users/jjwang/Downloads/wangboworkspace/print-client

# 3. 用 PM2 启动（会自动守护进程、自动重启）
pm2 start client.js --name print-client --cwd /Users/jjwang/Downloads/wangboworkspace/print-client

# 4. 保存进程列表（开机自启）
pm2 save

# 5. 设置开机自启
pm2 startup
```

## PM2 常用命令

```bash
pm2 list                    # 查看进程状态
pm2 logs print-client       # 查看日志
pm2 restart print-client    # 重启
pm2 stop print-client       # 停止
pm2 delete print-client     # 删除
pm2 monit                   # 实时监控
```

## 如果用传统方式（不推荐）

```bash
cd /Users/jjwang/Downloads/wangboworkspace/print-client
nohup node client.js > client.out.log 2>&1 &
echo $! > client.pid
```

## 日志位置

```
print-client/
├── logs/
│   └── client-YYYY-MM-DD.log   # 自动分日期
├── client.out.log              # stdout/stderr（传统方式）
└── client.pid                  # 进程 ID（传统方式）
```

## 日志查看

```bash
# 实时查看
tail -f /Users/jjwang/Downloads/wangboworkspace/print-client/logs/client-$(date +%Y-%m-%d).log

# 搜索错误
grep ERROR /Users/jjwang/Downloads/wangboworkspace/print-client/logs/client-*.log
```
