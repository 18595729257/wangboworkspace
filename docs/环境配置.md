# 环境配置

## 服务端（print-miniprogram-admin）

在服务器 `/opt/print-miniprogram-admin/` 下创建 `.env` 文件：

```bash
# 数据库
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=
DB_NAME=print_admin
DB_POOL_SIZE=20

# Redis（可选，不配置则缓存功能跳过）
# REDIS_HOST=127.0.0.1
# REDIS_PORT=6379
# REDIS_PASSWORD=
# REDIS_DB=0

# 微信小程序
WX_APPID=wx749cd8c41284e88f
WX_APPSECRET=your_appsecret_here

# JWT
JWT_SECRET=your-secret-key-change-in-production
JWT_EXPIRES_IN=7d

# 服务
PORT=3000
NODE_ENV=production
```

## 客户端（print-client）

在本地 `print-client/config.js` 中配置：

```javascript
module.exports = {
  API_URL: 'https://xinbingcloudprint.top',
  WS_URL: 'wss://xinbingcloudprint.top/ws/printer',
  CLIENT_ID: 'print-client-1',     // 每台电脑唯一 ID
  DOWNLOAD_DIR: './downloads',
  LOG_DIR: './logs',
  KEEP_FILES: false,
  POLL_INTERVAL: 15000,
}
```
