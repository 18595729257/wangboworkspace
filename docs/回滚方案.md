# 回滚方案

## 服务端回滚

```bash
# 1. 查看备份
ssh root@39.104.59.201 "ls -lh /opt/print-miniprogram-admin/server.js.backup.* | tail -5"

# 2. 回滚（选一个备份文件）
ssh root@39.104.59.201 "cp /opt/print-miniprogram-admin/server.js.backup.20260408-230000 /opt/print-miniprogram-admin/server.js"

# 3. 重启
ssh root@39.104.59.201 "pm2 restart print-admin"

# 4. 检查
ssh root@39.104.59.201 "pm2 logs print-admin --lines 20"
```

## 客户端回滚

```bash
cd /Users/jjwang/Downloads/wangboworkspace/print-client

# 查看 git 历史
git log --oneline | head -5

# 回滚到指定版本
git reset --hard <commit-id>

# 重启客户端
pm2 restart print-client
```

## 重构前完整备份（一次性）

如果需要回滚整个重构前状态，可以用 git：

```bash
cd /Users/jjwang/Downloads/wangboworkspace
git add -A
git commit -m "refactor: 架构重构 - 拆分路由和服务"

# 如果要回到重构前
git log
git reset --hard <旧commit-id>
```
